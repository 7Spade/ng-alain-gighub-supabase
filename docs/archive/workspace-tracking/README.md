## üéØ Archive Criteria

Documents are archived when they meet one or more of these criteria:

1. **Completed**: The document represents a completed phase or task
2. **Superseded**: The document has been replaced by a newer version or different approach
3. **Historical**: The document is valuable for historical context but not for current work
4. **Consolidated**: The information has been consolidated into other living documents
