# 封存檔案索引

本檔案列出 `docs/archive/` 中的主要封存文件，並提供簡短說明以便快速查找。

- `ARCHITECTURE_VISUAL_COMPARISON.md` — 架構視覺化比較，對比不同部署或設計方案的圖表與差異分析，便於決策與展示。
- `BLUEPRINT_REFACTORING_SUMMARY.md` — 藍圖重構總結，記錄重構範圍、重要變更點與影響評估。
- `blueprint-vertical-slice-architecture-analysis.md` — 垂直切片架構分析，逐層 (vertical slice) 分析功能從前端到後端的完整實作流程與依賴。
- `MULTI_TENANT_SAAS_CORE_DESIGN.md` — 多租戶 SaaS 核心設計，定義租戶隔離、資料分層、認證與 RLS 策略等核心架構決策。
- `MULTI_TENANT_SAAS_STATUS_REPORT.md` — 多租戶 SaaS 狀態報告，記錄專案進度、風險與下一步建議（階段性報告）。
- `TASK_LIST.md` — 歷史任務清單，包含已完成與已封存的任務摘要與對應追蹤紀錄。
- `TASK_NOW.md` — 當前或歷史的即時任務追蹤檔（封存副本），供查閱過往執行狀態與時間點。
- `README.md` — 本目錄說明與封存原則（請參閱此檔了解組織方式與檢索方法）。
